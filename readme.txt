Bablic is a localization service on the cloud. It will transform any site into a multi-lingual in minutes.
No changes in code or infrastructure required.
This plugin will inject Bablic snippet code into your html pages.

== Description ==

Save time & money, and give the best localization solution to your site.
This plugin will integrate your site with your Bablic account. If you don't have an account, you can create a free one at http://www.bablic.com

Bablic can transform any website, regardless of the server-side implementation, to a localized site. Bablic will automatically detect 'un-localized' text and send it to translation.
Bablic will send content to professional translators, or provide you with the option to translate yourself, directly on top of your site, the best user-interface your translators can ask for.

Bablic support bi-directional languages, CSS modifications in specific locales, image replacement, and much more.

To read more, on register for a free Beta account:
http://www.bablic.com

* This plugin sends and receives content from your site to our server.

== Installation ==

1. Upload `Bablic.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In plugin settings -> Type in your Bablic ID and check the 'Activated' box 

== Frequently asked questions ==

= How do I retrieve a Bablic ID =

Register for a free account on http://www.bablic.com, list your site. You'll be redirected to Bablic Backoffice page, the last part of the URL path, is your site's Bablic ID
